Default Flex

Try resizing the green rectangle by dragging the bottom right corner to the left.

You'll notice that the three sections will resize, with each one taking up more of the width based on how much text is in each box.

No fair! I'm so small!


Each box will be evenly sized

Try resizing the long green rectangle by dragging the bottom right corner to the left.

You'll notice that now the three sections will all be the same size, regardless of how much text is in each box.

That's much better! Although if you make us too small then you have to scroll left and right :(

Each box will be evenly sized and wrap if too narrow

Try resizing the long green rectangle by dragging the bottom right corner to the left.

You'll notice that now the three sections will be the same size and will wrap to the next line if they don't fit.

Yay! No more scrollbar. Although the text still gets really narrow and hard to read before it wraps.

Each box will be evenly sized and wrap if any box would be less than 100px wide without wrapping.

Try resizing the long green rectangle by dragging the bottom right corner to the left.

You'll notice that now the three sections will be the same size and will wrap to the next line if they don't fit.

Yay! Now the text doesn't get too narrow before it wraps.

Carousel with Horizontal Scroll Snapping

Try resizing the long green rectangle by dragging the bottom right corner to the left.

You'll notice that now the three sections will be the same size and will stay a minimum of 200px wide. If there is not enough space inside the parent div, then a horizontal scrollbar will appear.

Yay! Now I'm in a carousel!
