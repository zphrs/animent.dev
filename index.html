Animate, Experiment, Develop: Learn web development in three stages

Level 1: Animate

Hand code SVG images, and then animate those images to create stunning animations.

Level 2: Experiment

Use JavaScript with HTML and CSS to create an engaging and interactive experiment: a game, a microapp, or a web toy.

Level 3: Develop

Develop complete applications with the React framework, complete with a NodeJS backend.

About

This site was made to teach web design. For each level there is editable example code. You can edit the example code and see a preview of the page with your edits, but any edits you make will disappear once you refresh the page. If you want to save your changes I recommend copying and pasting your edits into replit.com, Visual Studio Code, or a different text editor.
